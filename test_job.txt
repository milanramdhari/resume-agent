Requirements
The position requires development of software for thermal imaging camera systems for ADAS and automotive applications, and the ability to lead various aspects of software development related to Thermal Camera development such as GUI interface, thermal sensor calibration software, image quality enhancement software algorithms, and software traceability in server.
Responsibilities
Review and analyze product requirements (including requirements from customer and supplier) and decompose the requirements to the lowest component level engineering.
Develop C++ engineering applications: Design and development of software used in the testing and validation of infrared Cameras.
Applications to be written in Visual C++ using Microsoft Foundation Classes and run in Windows 11 and Ubuntu environments.
Expand image analysis capabilities of the existing application.
Develop automated tests to exercise specific function and properties of the thermal camera.
Write applications code using object oriented concepts and data structures.
Revision control/ software Configuration Management in Team Foundation Server.
Generate new and maintain existing installer package for deployment of all software applications.
Coordinate with users the deployment of new software releases.
Write and deliver test plan to validate new software releases.
Conduct problem solving with identified issues in various testing setups and respective products.
Conduct risk analysis such as DFMEA/ DRBFM for complex camera systems.
Provide technical interface and input to manufacturing, quality, purchasing, customer, and suppliers to achieve successful completion of component, product and system design objectives within time and cost constraints.
Develop test methods, engineering reports, fixture/prototype designs and test execution support for product development
